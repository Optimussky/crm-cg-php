## Create a new respository on command line

git init
git add .
git commit - "first commit"
git branch -M main
git remote add origin https://github.com/Optimussky/crm-cg-php.git
git push -u origin main


## Push an existing repository from the command line

git remote add origin https://github.co/Optimussky/crm-cg-php.git
git branch -M main
git push -u origin main
